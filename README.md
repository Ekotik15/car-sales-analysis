# Прогнозирование продаж автомобилей

Проект по машинному обучению для прогнозирования объемов продаж автомобилей на основе их характеристик.

##  О проекте

Этот проект решает задачу регрессии - прогнозирование объемов продаж автомобилей (в тысячах единиц) на основе технических характеристик, цены и других параметров.

###  Цели проекта

- Построить модель машинного обучения для прогнозирования продаж
- Провести подбор гиперпараметров и оптимизацию модели
- Обеспечить интерпретируемость результатов
- Создать воспроизводимое решение
